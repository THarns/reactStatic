-open bash terminal
-git init
-npm init -y
-install devDependency packages from appropriate text file (NPM devDependencies.txt)
-update npm scripts 
-build (npm run build:dev in terminal)
-run dev-server (npm run start:dev in terminal) open web browser and navigate to your IP address:8080. when files are saved, the browser should automatically refresh with the changes.

*If snap.svg is needed use snapsvg-cjs package instead to avoid imports-loader hack
